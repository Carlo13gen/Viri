This directory contains Java and C++ source files of a simple Wrapper that allows to use the GDToolkit library in a Java application.

-----------------------------------------------------------------------
COMPILE THE WRAPPER AND THE LIBRARY 
-----------------------------------------------------------------------

Use compile.bat (Microsoft Windows) or compile.sh (Linux) to compile the java source files and to produce a JAR file.

Compile the GDToolkit library together with the files:
 	- java_wrapper\cpp\src\GdtJavaWrapper.cpp
	- java_wrapper\cpp\include\GdtJavaWrapper.h
	- java_wrapper\cpp\include\GdtJavaWrapper_WrapperImplementation.h (the last file is automatically generated by compile.bat or compile.sh)


-----------------------------------------------------------------------
USING GDTOOKIT IN A JAVA APPLICATION 
-----------------------------------------------------------------------
 
To use the GDToolkit library in a java application, add the following lines to the Java source code:

	import GdtJavaWrapper.*;

In the class containing the main, out of the scope of the methods: 
	static { System.loadLibrary("GdtWrapper"); }


-----------------------------------------------------------------------
EXAMPLE 
-----------------------------------------------------------------------
It is possible to use the following commands in a java source file:

		Graph g=new Graph();
		
		Node n1=g.newNode();
		Node n2=g.newNode();
		Node n3=g.newNode();

		g.newEdge(n1, n2);
		g.newEdge(n1, n3);
		
		g.isPlanar();   //Planarity testing of graph g


		OrthogonalDrawer D=new OrthogonalDrawer(g);  //Orthogonal drawing of graph g
		
		D.getNodeRectangle(n1);
